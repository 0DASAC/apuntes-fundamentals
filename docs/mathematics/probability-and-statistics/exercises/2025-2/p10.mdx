$$
f(x) =
\begin{cases}
  0.2 + \frac{x}{15}, & \text{si } -3 < x < 0 \\
  0.2 -  \frac{x}{35} & \text{si } 0 < x < 7 \\
  0 & \text{en otro caso}
\end{cases}
$$

¿Cuál de las siguientes alternativas es el valor **MÁS CERCANO** a la esperanza o valor esperado de 𝐶?

a) 0

b) 1

c) 1.33
