# Pregunta 2

$$
\frac{x-2}{5}=\frac{y+5}{1}=\frac{z+1}{3}
$$
está dada por:

a) $7 x-4 y+2 z=0$

b) $5 x+y+3 z=0$

c) $2 x-5 y-z=0$

d) El plano no se encuentra determinado
