# Pregunta 3

El volumen de $\Omega$ es:

a) $\frac{16}{3}\left(1-\frac{1}{\sqrt{2}}\right) \pi$

b) $\frac{16}{3} \pi$

c) $\frac{8}{3}\left(1-\frac{\sqrt{3}}{2}\right) \pi$

d) $\frac{16}{3}\left(1-\frac{\sqrt{3}}{2}\right) \pi$
