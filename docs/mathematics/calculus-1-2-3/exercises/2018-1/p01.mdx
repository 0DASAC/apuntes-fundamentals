# Pregunta 1

c) $\frac{1}{2} \sqrt{2} \arctan \left(\frac{1}{2} \sqrt{2} x^{\frac{1}{10}}\right)+C$

d) $\frac{1}{2} \sqrt{2} \arctan \left(\frac{1}{2} \sqrt{2} x^{\frac{1}{5}}\right)+C$
