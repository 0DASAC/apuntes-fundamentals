# Pregunta 5

<div className="exercise">
Considere la función $𝑔: ℝ^2 → ℝ$ dada por:
$$
𝑔(𝑥, 𝑦) = sin (𝑥) (cos (𝑦))^2
$$
Y su derivada direccional en la dirección de un vector unitario arbitrario $\hat{u}$.
¿En cuál de los siguientes puntos, la derivada direccional de 𝑔 es **NULA** para todo valor posible de
$\hat{u}$?

a) (𝑥, 𝑦) = (𝜋/3, 𝜋/2)

b) (𝑥, 𝑦) = (𝜋/2, 𝜋/3)

c) (𝑥, 𝑦) = (0, 0)

d) (𝑥, 𝑦) = (−𝜋, 𝜋)

</div>

<MDXDetails>
<summary>Solución propuesta</summary>

<div className="exercise">

**Respuesta:**

a) (𝑥, 𝑦) = (𝜋/3, 𝜋/2)

Aún no hay solución propuesta 🥲
</div>
