# Pregunta 25

<div className="exercise">
La figura muestra un núcleo de hierro de sección transversal $A$ al cual se le conecta un solenoide. Por este núcleo circula un flujo magnético de forma $\phi=\phi_0 \sin (t)$.

<Image
  src="science/electromagnetism/exercises/2023-2/p25/image.png"
  alt="Pregunta 25"
  width="40%"
/>

Si en el extremo del solenoide se mide el voltaje según la figura, ¿cómo será la forma del voltaje?

a) $V=-\phi_0 A \cos (t)$

b) $V=\phi_0 \cos (t)$

c) $V=-5 \phi_0 A \cos (t)$

d) $V=5 \phi_0 \cos (t)$
</div>
