# Pregunta 35

<div className="exercise">
El cambio de entropía de un sistema depende de:

a) La cantidad de masa transferida.

b) El calor transferido.

c) El cambio de temperatura.

d) El cambio de presión y volumen.
</div>

<MDXDetails>
<summary>Solución propuesta</summary>

<div className="exercise">

**Respuesta:**

b) El calor transferido.

**Explicacion:**

La entropía es una propiedad termodinámica que mide el grado de desorden o la cantidad de energía no disponible para realizar trabajo. En términos matemáticos, su cambio se relaciona directamente con el calor transferido de forma reversible.
</div>
