# Pregunta 37

<div className="exercise">
Como ingeniero le han pedido diseñar un sistema para aprovechar una corriente de vapor que viene
a 200 °C para tratar de generar trabajo. El calor puede ser descargado a un reservorio frío que se
encuentra a 80 °C. Le han planteado que, si la eficiencia máxima que se podría llegar a obtener de
este sistema es menor al 50%, la idea debería descartarse.

¿Qué pasará con el proyecto?

a) No se puede calcular la eficiencia máxima con la información dada.

b) Se debe descartar.

c) Debe continuar.

d) Ninguna de las anteriores.

</div>

<MDXDetails>
<summary>Solución propuesta</summary>

<div className="exercise">

**Respuesta:**

b) Se debe descartar.

**Desarrollo:**

Sabemos que cualquier sistema tiene que ser menos eficiente que el sistema de carnot. por lo que podemos calcular su eficiencia máxima utilizando la fórmula de Carnot:

$$\eta_{max} = 1 - \frac{T_L}{T_H}$$


$$\eta_{max} = 1 - \frac{353,15}{473,15} = 0.2536$$

Por lo tanto, la eficiencia máxima que se podría obtener de este sistema es del 25%, lo que es menor al 50% planteado por el ingeniero. Por lo tanto, la idea debería descartarse.
</div>
