# Pregunta 38

<div className="exercise">
En termodinámica se designa como proceso adiabático aquel sistema que no intercambia calor con su entorno o alrededores.

Luego, el cambio de entropía en un proceso adiabático será:

a) Mayor a cero.

b) Igual a cero.

c) Mayor o igual a cero.

d) Menor o igual a cero.
</div>

<MDXDetails>
<summary>Solución propuesta</summary>

<div className="exercise">

**Respuesta:**

c) Mayor o igual a cero.

**Expliacion:**

Porque no igual a cero? porque el proceso puede tener irreversibilidades como por ejemplo roces internos o turbulencias.
</div>
