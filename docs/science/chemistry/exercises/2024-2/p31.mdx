# Pregunta 31

<div className="exercise">
Indique cuál de las siguientes disoluciones generará una mayor cantidad de iones, si todas se encuentran disueltas en un mismo volumen de agua:

a) Cloruro de sodio, $\mathrm{NaCl}, 0,5 \mathrm{~mol} / \mathrm{L}$.

b) Glucosa, $\mathrm{C}_6 \mathrm{H}_{12} \mathrm{O}_6 1,3 \mathrm{~mol} / \mathrm{L}$.

c) Ácido clorhídrico, $\mathrm{HCl}, 1,0 \mathrm{~mol} / \mathrm{L}$.

d) Hidróxido de sodio, $\mathrm{NaOH} 0,7 \mathrm{~mol} / \mathrm{L}$.
</div>

<MDXDetails>
<summary>Solución propuesta</summary>

<div className="exercise">

**Respuesta:**

c) Ácido clorhídrico, $\mathrm{HCl}, 1,0 \mathrm{~mol} / \mathrm{L}$.

**Expliacion:**

Nos importa mirar principalmente la concentracion porque a mayor concentracion mayor cantidad de moles (solo si es el mismo volumen). ademas nos interesa ver cuantos iones livera cada uno porque algunas especies pueden liverar mas de uno.

En este caso todas las especies liveran la misma cantidad de iones por lo que nos basta con revisar la que tenga mayor concentracion. En este caso la c)
</div>
