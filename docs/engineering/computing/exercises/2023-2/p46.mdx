# Pregunta 46

<div className="exercise">
A continuación, se muestra un pseudocódigo de un programa que utiliza distintos tipos de datos.
Asuma que `a%b` entrega el resto de la división entera entre dos enteros `a` y `b`. Por ejemplo, `5%2`
entrega el valor `1`.

```
1.    a = 3
2.    b = 15.0
3.    c = VERDADERO
4.    d = a - b
5.    e = d > 0
6.    f = b%2 == 1
7.    g = e AND f
```

Al terminar de ejecutarse el programa anterior, ¿qué valores quedan almacenados en las variables `e`, `f` y `g`?

a) `e = -12.0, f = FALSO, g = FALSO`

b) `e = FALSO, f = FALSO, g = FALSO`

c) `e = FALSO, f = VERDADERO, g = FALSO`

d) `e = 15.0, f = FALSO, g = VERDADERO`
</div>
