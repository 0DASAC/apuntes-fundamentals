# Pregunta 47
Considere el siguiente programa.

Recuerda que la función **len(lista)** retorna la cantidad de elementos que tiene la lista lista,
que **len(texto)** retorna la cantidad de caracteres que tiene el string **texto**, y que **lista$[i]$**
accede al elemento de lista que se encuentra en el índice **i**.

Supón que **lista** ya está inicializada como una lista de enteros.

<Image
  src="engineering/computing/exercises/2025-2/p47/image.png"
  width="65%"
/>

Si **lista** $= [10,4,423,1523,55,9999]$ , ¿qué valor queda almacenado en la variable
**nueva_lista**?

a) $[9999, 55, 1523, 423, 4, 10]$

b) $[4, 2, 4, 3, 1, 2]$

c) $[0, 4, 3, 3, 5, 9]$

d) $[2, 1, 3, 4, 2, 4]$
