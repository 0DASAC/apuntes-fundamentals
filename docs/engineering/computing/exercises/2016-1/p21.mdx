# Pregunta 21

<div className="exercise">
Observe el siguiente código donde se busca ver si un número `num` positivo de dos dígitos cumple una condición o no (`cumple` es `True` o `False`). Asuma que `a//b` entrega la división entera entre `a` y `b`:

```python
a = num - 10*(num//10)
b = num//10
cumple = (a+b + a*b == num)
```

¿Cuál es la condición de `num` para que `cumple` sea `True`?

a) Diez veces la unidad del número más la multiplicación del número por sus dígitos debe ser igual al número.

b) La suma de las unidades más la multiplicación de ellas debe ser igual al número.

c) No hay condición, pues `a` siempre es 0.

d) La suma de sus dígitos más la multiplicación de ellos debe ser igual al número.
</div>
