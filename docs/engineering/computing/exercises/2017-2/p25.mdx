# Pregunta 25

<div className="exercise">
Considere el siguiente código, donde la variable `largo` ya está definida, con valor `9` (`largo = 9`).

```
lista = [12,10,3,5,1,5,22,0,12]
largo = 9
minimo = 100
i = 0
while i < largo:
  if minimo > lista[i]:
    minimo = lista[i]
  i = i + 1
```

En algún momento de la ejecución del programa, `minimo` tiene valor `1` (`minimo = 1`). En ese momento, ¿qué valor puede tener `i`?

a) `0`

b) `1`

c) `4`

d) `8`
</div>
